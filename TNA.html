TNA

Full e-commerce web app

Web Applications exam project AA. 2019-2020.

The project consists of the creation of a web app for the management of an e-commerce website with user registration, secure login, many db operations, product search functions, categories filters, profile management, recent orders, shopping cart and organization of available objects.

Technologies used:

  • HTML5
  • CSS3
  • JavaScript (JS)
  • jQuery
  • Semantic UI
  • Node.js
  • Express
  • MongoDB
  • npm
